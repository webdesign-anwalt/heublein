== Description ==

Extra set of shortcodes for Shortcodes Ultimate. [Visit project page](http://gndev.info/sue).


= Requirements =

* PHP 5.1+
* WordPress 3.5+
* Shortcodes Ultimate 4.4+


= Support =

* [Knowledge base](http://gndev.info/kb)
* [Customer support](http://gndev.info/cs)


== Installation ==

This add-on is distributed as an regular zipped plugin. So, you can install it like any other plugins. Navigate to Dashboard -> Plugins -> Add New -> Upload. Then select downloaded zip-archive. [Read more about plugin installation](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
[Demo video](http://www.youtube.com/watch?v=FSy8hCOzLX0).
